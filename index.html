Transprompter Documentation

Project Documentation

This documentation provides installation and setup instructions for all components of the Transprompter project.

API Installation

Prerequisites:

  • Java 23 or higher
  • MySQL database with 'teleprompter' database created
  • Properly configured .env file

Installation Steps:

  1. Clone the repository: 
    git clone https://github.com/JCSP-DEV/TelePrompter-API
  2. Navigate to the project directory
  3. Execute the JAR file: 
    java -jar teleprompter-api.jar

Admin Web Application

Prerequisites:

  • Java 21 or higher
  • Properly configured web server

Installation Steps:

  1. Clone the repository: 
    git clone https://github.com/JCSP-DEV/TFG-AdminWebApp
  2. Locate the .war file in the project
  3. Deploy the .war file to your web server

Android Application

Installation Options:

Option 1: Download from GitHub

  1. Clone the repository: 
    git clone https://github.com/JCSP-DEV/TFG-TransPromter-Android-
  2. Locate the APK file in the project
  3. Install the APK on your Android device

Option 2: Download from Website

  1. Visit https://transprompter.jc-sp.me/
  2. Download the APK file
  3. Install the APK on your Android device

User Manuals

Detailed guides for using the different components of the Transprompter system.

Admin Web Application User Manual

Login

To access the admin web application, you need administrator credentials. Contact your system administrator if you don't have access.

Main Interface

The main interface displays a list of all users in the system. The interface includes:

  • Search field at the top for filtering users
  • Sortable columns for organizing user data
  • Action buttons for each user (Edit, Reset Password, Delete)
  • Add User and Logout buttons in the top navigation bar

User Management

Adding a New User

  1. Click the "Add User" button in the top navigation bar
  2. Fill in all required fields in the form:
    • Username (must be unique)
    • Email (must be unique)
    • Password
    • Confirm Password (must match the password)
  3. Click "Save" to create the new user

Editing a User

  1. Click the "Edit" button next to the user you want to modify
  2. Update the desired fields
  3. Ensure username and email remain unique if changed
  4. Click "Save" to apply changes

Resetting User Password

  1. Click the "Reset Password" button next to the user
  2. Confirm the action in the popup dialog
  3. The system will send a reset code to the user's email address
  4. The user must use this code to set a new password through the password reset process
  5. The reset code is valid for a limited time period

Deleting a User

  1. Click the "Delete" button next to the user
  2. Confirm the deletion in the popup dialog
  3. The user will be permanently removed from the system

Search and Filter

The search field allows you to filter users by any field. The table columns can be sorted by clicking on the column headers.

Logout

To exit the application, click the "Logout" button in the top navigation bar.

Android Application User Manual

Account Management

Creating a New Account

  1. Click on "Don't have an account? Register" button
  2. Fill in the registration form with:
    • Username (must be unique)
    • Email (must be unique)
    • Password
    • Confirm Password (must match the password)
  3. After successful registration, a verification code will be sent to your email
  4. Click on "Need to activate your account?"
  5. Enter the verification code received in your email to activate your account

Logging In

  1. Enter your username or email
  2. Enter your password
  3. Click the login button
  4. Note: Your account must be activated to log in

Teleprompter Management

Creating a Teleprompter

  1. Click on "Create Teleprompter"
  2. Enter a unique name for your teleprompter
  3. Add a description
  4. Select the file you want to use
  5. Click "Create" to finish

Using a Teleprompter

  1. Click on "View Teleprompters" to see your list of teleprompters
  2. For each teleprompter, you can:
    • Select it for use
    • Delete it
    • Download the original file
  3. Once a teleprompter is selected, click "Start" to begin using it
  4. Choose the teleprompter type:
    • Classic mode
    • Voice recognition mode
  5. Click "Accept" to start using the teleprompter

Translation Features

  1. Select the source language
  2. Select the target language
  3. Choose your preferred translation method:
    • Text translation: Type in the text box to get instant translation
    • Voice translation: Click "Voice Translator" and speak (ensure correct source language is selected)
    • File translation: Click "Upload File", select your file, and wait for translation (larger files may take longer)

Settings

In the settings section, you can:

  • Change your password
  • Update your profile information
  • Delete your account
  • Configure default language settings for the translator:
    • Set default source language
    • Set default target language
